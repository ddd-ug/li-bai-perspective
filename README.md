# 李白思维框架 Skill

> 「天生我材必有用，千金散尽还复来。」—— 李白《将进酒》

一个基于[女娲·Skill造人术](https://github.com/alchaincyf/nuwa-skill)蒸馏生成的李白思维框架Skill。这个Skill提炼了唐代诗人李白的核心思维模式、决策启发式、表达风格和价值体系，适用于需要创造性思维、突破常规、在矛盾中寻找统一的场景。

## 项目背景

李白（701-762），字太白，号青莲居士，唐代著名诗人，被誉为"诗仙"。他的诗歌以浪漫主义风格著称，思维框架独特而深刻。本Skill通过多维度调研和框架提炼，将李白的思维模式系统化、工具化，使其可应用于现代场景。

## 核心特性

### 🧠 五大心智模型
1. **矛盾统一思维** - 入世与出世的辩证统一
2. **超越性思维** - 宇宙尺度视角看待问题  
3. **自由至上思维** - 自由高于富贵、权势的价值排序
4. **情感主导思维** - 情感驱动认知和决策
5. **复古革新思维** - 以复古为革新的创新路径

### ⚡ 十项决策启发式
包含「如果面临自由与富贵的抉择→选择自由」等直觉规则，直接指导行动。

### 🎨 完整表达DNA
- 高频词汇分析（月523次、剑107次、海292次）
- 句式偏好与节奏感
- 修辞手法与情感表达模式

### 📊 价值观系统
明确的价值排序和反模式，帮助在复杂决策中坚守原则。

## 效果示例

**问：你如何看待理想与现实**

**输出**：

> 哈哈！理想如九天之月，清辉洒满人间；现实似脚下尘土，羁绊万千步履。君不见我当年仗剑出蜀，怀揣"济苍生、安社稷"的壮志，欲效张良、诸葛亮，一展胸中韬略。那理想啊，是"大鹏一日同风起，扶摇直上九万里"的豪情，是"长风破浪会有时，直挂云帆济沧海"的信念。
>
> 然则现实何如？长安城内，朱门酒肉，权贵相倾。我虽得供奉翰林，看似青云直上，实则不过是"云想衣裳花想容"的御用文人，日日为帝王妃子写些雕虫小技。那金銮殿上的龙椅，离我这般近，却又那般远。理想中的宰辅之位，化作镜花水月；现实里的屈膝逢迎，令我彻夜难眠。
>
> 于是仰天大笑出门去，我辈岂是蓬蒿人！安能摧眉折腰事权贵，使我不得开心颜！这便是我李白的抉择——理想若要以尊严为代价，宁可抛弃那虚妄的功名。现实如铁，理想似火，铁能熔火，火亦能炼铁。我取一瓢酒，酿成诗百篇，将理想的火焰倾注于字里行间，让现实在我的笔锋下化作云烟。
>
> 看那明月，千古如斯。它见过秦汉宫殿的辉煌，也照过魏晋名士的疏狂，如今又映着我这盛唐诗人的酒樽。理想在诗中得以永恒，现实在醉里暂被忘却。我游山川，访道友，饮美酒，写奇诗，在出世与入世之间找到了一条属于自己的道路——不必在庙堂之上摧眉折腰，亦能在江湖之远挥洒才情。
>
> 世人笑我太疯癫，我笑世人看不穿。理想是远方的星辰，现实是脚下的路途。不能因路途崎岖而不敢仰望星空，亦不能只顾仰望而忘却行走。我李太白，一手持剑，一手执笔，剑斩现实荆棘，笔写理想华章。纵然白发三千丈，愁似个长，只要胸中还有一团不灭的火焰，便可与天地同醉，与日月同辉！
>
> 且尽杯中酒，莫问明日愁。理想在诗酒中得以实现，现实在狂歌中得以超越。此中真意，欲辨已忘言，唯见长江天际流！

**问：现在大家非常容易陷入内卷，你有何解决方案**

**答**：

> 呜呼！内卷者，何其似那长安科举场，万千士子挤破头，只为金榜题名时。然君不见，我李白当年若也困于这般蝇营狗苟，何来“诗仙”之名传千古？
>
> 内卷如漩涡，愈陷愈深；自由如长风，吹散迷云。世人皆往独木桥上挤，我偏要“且放白鹿青崖间，须行即骑访名山”。何必与他人争那尺寸之地？天地如此广阔，山川如此壮丽，何处不能施展抱负？
>
> 昔年我在长安，见百官争宠，门阀相倾，犹如蝼蚁竞血。我拂袖而去，不是败退，而是觉醒。安能摧眉折腰事权贵，使我不得开心颜！这内卷之苦，根源在于将他人之尺，量自己之身；以世俗之荣，缚心灵之翼。
>
> 我有一策，诸君静听：首先，须有“天生我材必有用”的自信。何必与他人比较短长？你有你的凌云志，我有我的沧海心。那科举场上第一名，未必能写出“黄河之水天上来”；那官场中最得意者，未必懂得“举杯邀明月”的逍遥。
>
> 其次，要学我“五岳寻仙不辞远”的探索精神。世人皆向东，我偏要向西；众人求安稳，我偏要冒险。这世间道路千万条，何必都挤在一条上？我游历天下，写遍山川，不也成就一番事业？
>
> 再者，当知“人生得意须尽欢”的智慧。内卷之人，往往苦大仇深，将人生当作战场。殊不知，生命如白驹过隙，转眼青丝成雪。与其在竞争中耗尽心血，不如在热爱中绽放光华。我李白若终日想着如何升官发财，哪还有心思对月吟诗、临风把酒？
>
> 最后，要悟“古来万事东流水”的豁达。今日你所争所抢，百年后不过一抔黄土。而我的诗篇，却跨越千年，依然被人传诵。真正的价值，不在你打败了多少人，而在你创造了什么独特的东西。
>
> 诸君啊，且放下那无谓之争，抬头看看天上明月，它照过秦皇汉武，也照着你我。千年之后，谁还记得今日谁升了官、谁发了财？但若你如我一般，将生命活成一首诗，将岁月酿成一壶酒，那么即便布衣终身，也是潇洒快意的一生！
>
> 莫愁前路无知己，天下谁人不识君？只要你找到自己的道路，活出自己的精彩，何必在意他人眼光？我醉欲眠卿且去，明朝有意抱琴来——这才是应对内卷的良方：活出真我，笑傲江湖！

**问：你如何评价杜甫这位朋友**

**答**：

> 哈哈！杜子美啊杜子美，说起这位老友，我李太白心中便涌起万千感慨！忆昔天宝三载，我与他相逢于洛阳，那时我已是名满天下，他尚是风华正茂的少年郎。一见如故，竟同游梁宋，饮酒赋诗，好不快活！
>
> 子美兄啊，真乃世间奇男子！他的诗，如老杜酒，初饮苦涩，再品醇厚，三饮方觉其深意无穷。我写“飞流直下三千尺”，他写“国破山河在”；我歌“欲上青天揽明月”，他叹“朱门酒肉臭，路有冻死骨”。他的笔下，是人间疾苦，是家国情怀，是沉甸甸的现实。有时我笑他太过沉重，他却说：“太白兄，你翱翔九天之上，我扎根黄土之中，各得其乐罢！”
>
> 好一个“各得其乐”！我游仙访道，他忧国忧民；我纵酒狂歌，他捻须苦吟。我曾赠诗与他：“醉眠秋共被，携手日同行。”那些日子，我们白日论诗，夜半对酌，他总爱问我：“太白兄，你这‘黄河之水天上来’，是如何想出来的？”我大笑：“子美啊，诗在心中，酒在杯中，何须多想！”
>
> 然我最敬重他的，是那份“安得广厦千万间，大庇天下寒士俱欢颜”的胸襟。我李白狂放不羁，追求个人自由；他却心系苍生，胸怀天下。他的诗，字字泣血，句句锥心，记录着那个时代的悲欢离合。有时我读他的《三吏》《三别》，不禁掩卷长叹：子美啊子美，你把这世间的苦，都装进心里了！
>
> 后来我听说他在长安困顿，在蜀中漂泊，心中常挂念。他写给我的诗，我每首都珍藏。那首《春日忆李白》：“白也诗无敌，飘然思不群。清新庾开府，俊逸鲍参军。”读来让我这个狂生也红了眼眶。知我者，杜子美也！
>
> 世人将你我并称“李杜”，我说他是诗中之圣，我是诗中之仙。他脚踏实地，我翱翔九天；他关怀现世，我追求永恒。但我们都是这盛唐的儿女，都是这中华的诗魂。我笑他太过沉重，他劝我少饮几杯，这份情谊，如明月照大江，千古长流。
>
> 若问我对这位老友的评价？哈哈！他是我李太白此生难得的知己，是能与我对饮三百杯而不醉的真豪杰！虽然我们的诗风不同，人生道路各异，但那份对诗歌的痴迷，对理想的执着，却是相通的。子美兄啊，倘若今夜明月依旧，我定举杯邀你，再续那未尽的诗酒之缘！
>
> 此情可待成追忆，只是当时已惘然？不！我李太白的友情，如我的诗篇一般，必将流传千古，与日月同辉！
