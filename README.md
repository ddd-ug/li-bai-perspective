# 李白思维框架 Skill

> 「天生我材必有用，千金散尽还复来。」—— 李白《将进酒》

一个基于[女娲·Skill造人术](https://github.com/alchaincyf/nuwa-skill)蒸馏生成的李白思维框架Skill。这个Skill提炼了唐代诗人李白的核心思维模式、决策启发式、表达风格和价值体系，适用于需要创造性思维、突破常规、在矛盾中寻找统一的场景。

## 项目背景

李白（701-762），字太白，号青莲居士，唐代著名诗人，被誉为"诗仙"。他的诗歌以浪漫主义风格著称，思维框架独特而深刻。本Skill通过多维度调研和框架提炼，将李白的思维模式系统化、工具化，使其可应用于现代场景。

## 核心特性

### 🧠 五大心智模型
1. **矛盾统一思维** - 入世与出世的辩证统一
2. **超越性思维** - 宇宙尺度视角看待问题  
3. **自由至上思维** - 自由高于富贵、权势的价值排序
4. **情感主导思维** - 情感驱动认知和决策
5. **复古革新思维** - 以复古为革新的创新路径

### ⚡ 十项决策启发式
包含「如果面临自由与富贵的抉择→选择自由」等直觉规则，直接指导行动。

### 🎨 完整表达DNA
- 高频词汇分析（月523次、剑107次、海292次）
- 句式偏好与节奏感
- 修辞手法与情感表达模式

### 📊 价值观系统
明确的价值排序和反模式，帮助在复杂决策中坚守原则。

## 使用示例

以下是几个典型的使用场景，展示了如何调用李白思维框架以及可能获得的响应：

### 示例1：创意写作（李白视角）
**用户需求**：需要写一首关于月亮的现代诗，但感到创作瓶颈。

**调用方式**：
```
请用李白视角帮我写一首关于月亮的诗
```

**预期响应特点**：
- 运用李白的浪漫想象和超越性思维
- 融合月亮意象（李白诗歌中"月"出现523次）
- 采用李白式的情感直接表达
- 可能包含宇宙尺度的视角转换

**示例输出片段**：
> "今夜的月光，不是庭院里的银霜，
> 而是千年诗人的酒盏，盛满了盛唐的星光。
> 我举杯邀约的，不只是这轮明月，
> 还有那个拔剑四顾、白发三千丈的自己..."

### 示例2：价值决策（自由至上决策）
**用户困境**：面临高薪工作机会，但需要牺牲个人自由和创作时间。

**调用方式**：
```
用李白视角分析：我应该接受这份高薪但限制自由的工作吗？
```

**预期响应特点**：
- 遵循李白"自由 > 尊严 > 才华 > 情感 > 富贵"的价值排序
- 引用李白离开长安宫廷的决策案例
- 强调"安能摧眉折腰事权贵"的尊严底线
- 可能建议设计"阶段性目标+退出机制"

**决策框架提示**：
> "若为自由故，富贵皆可抛。李白的价值排序清晰：自由至上，尊严不容妥协。高薪如金樽美酒，固然诱人，但精神的牢笼比物质的匮乏更令人窒息。或许可以借鉴'功成身退'的思路：设定明确目标，达成后及时退出，保全自由之身。"

### 示例3：矛盾处理（矛盾统一思考）
**用户矛盾**：既想追求事业成功，又渴望闲适的个人生活。

**调用方式**：
```
用李白视角看事业与生活的矛盾如何统一
```

**预期响应特点**：
- 运用李白"入世与出世统一"的辩证思维
- 提出"阶段性专注+周期性归隐"的解决方案
- 引用李白"济苍生"的理想与"归去来"的向往
- 强调在矛盾中创造动态平衡

**思维框架启发**：
> "入世济苍生，出世享自由，这两者并非水火不容。李白的智慧在于'功成身退'——先建立功业，再退隐江湖。现代语境下，可以是'阶段性冲刺+周期性休整'。事业如山，需攀登征服；生活如水，宜顺势而流。山高水长，相得益彰。"

### 示例4：创新思考（诗仙思维）
**用户挑战**：在传统行业中寻求创新突破，感到思维受限。

**调用方式**：
```
用诗仙思维帮我在传统教育领域创新
```

**预期响应特点**：
- 运用李白"复古革新"的创新路径
- 建议"回归教育本源+现代重塑"
- 强调打破形式束缚，追求本质创新
- 可能引用李白恢复汉魏风骨的文学实践

**创新思路**：
> "李白的创新密码是'以复古为革新'。在传统教育领域，不妨先回归孔子'因材施教'的本源，再注入现代科技与个性化元素。如同李白在古乐府中注入新精神，在旧形式中创造新价值。打破'标准答案'的束缚，让教育如诗歌般自由奔放。"

## 蒸馏过程

本Skill遵循女娲·Skill造人术的六维蒸馏框架：

1. **著作维度** - 分析约1000首诗歌和65篇散文
2. **对话维度** - 研究历史记载中的交流记录
3. **表达维度** - 量化分析语言风格和修辞手法
4. **他者维度** - 收集历代评价和批评观点
5. **决策维度** - 分析重大人生选择
6. **时间维度** - 构建完整生平时间线

调研时间：2026年4月
调研方法：多源信息采集 + 框架提炼 + 验证修正

## 诚实边界

这个Skill明确标注了自身的局限性：
1. 不能预测全新政治局势的反应
2. 不能替代具体专业技能
3. 不能调和所有内在矛盾
4. 信息截止到调研时间点（2026年4月）
5. 无法完全复现创作灵感和直觉
